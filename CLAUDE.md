# Project Memory & Instructions

See @AGENTS.md for complete project context, design tokens, folder structures, and strict code quality guidelines.

## Development Commands

- `npm run dev` — Start the Next.js local development server
- `npm run build` — Build the production-optimized application
- `npm run start` — Run the production server locally
- `npm run lint` — Run ESLint check over the project logic

## Post-Edit Verification Workflow
After implementing any changes and before declaring a task done, you must execute:
1. `npm run lint`
2. `npm run build`
